üö© Complete Test Flow

 1. Create a Brand: 
 json
POST http://localhost:8083/api/brands
{
  "brandName": "Apple",
  "description": "Technology company",
  "isActive": true
}
 
Save `brandId` from response.

 2. Create a Category: 
 json
POST http://localhost:8083/api/categories
{
  "categoryName": "Smartphones",
  "description": "Mobile phones",
  "isActive": true
}
 
Save `categoryId` from response.

 3. Upload Product Images: 
 
POST http://localhost:8083/api/upload/images
[Select 3-4 product images]
 
Save image URLs from response.

 4. Create a Product: 
Use the brandId, categoryId, and image URLs from above steps in the create product request.

 5. Test Search: 
 
GET http://localhost:8083/api/products/search?query=iphone
 

 6. Test Get Product: 
 
GET http://localhost:8083/api/products/{productId}
 

 7. Test Update Product: 
 json
PUT http://localhost:8083/api/products/{productId}

  "basePrice": 949.99,
  "isFeatured": true
}
